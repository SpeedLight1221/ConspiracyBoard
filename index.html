Speedlight1221's Crazy Conspiracy Board: Web page

Welcome to Speedlight1221's Crazy Conspiracy Board: Web page! Made because deltarune chapters 3 & 4 came out and i needed to get my thought "in order". It could hvae been done in paint but i was too stubborn to do it that way. I hope you find it useful.

Version 1.1.1

Changes:

Increased board size again

Tools:

Click - Default tool, use when editing content of textboxes or when moving the board

Text - Creates a textbox.

Image - Creates an image box. Drag and drop an image to display in on the board

Pin - Creates a pin

String - Connects two pins with a line

Delete - Deletes object

Color - Open the color menu. Chosen color affects pins and strings

Everything is saved locally in your browser
